# BERTopic에서 임베딩 모델 및 풀링 전략에 따른 토픽 모델링 성능 비교 분석

본 저장소는 사전학습 언어 모델(**S-BERT**)과 대형 언어 모델(**LLaMA**)을 활용할 때, 다양한 풀링 전략(**CLS, Mean, Max, Dual Pooling**)이 BERTopic 기반의 토픽 모델링 성능(Coherence, Diversity)과 자원 효율성에 미치는 영향을 실증적으로 비교·분석한 연구 코드 및 안내를 포함하고 있습니다.

---

## Ⅰ. 연구 개요 (Overview)
자연어 처리(NLP) 분야에서 비정형 텍스트를 밀집 벡터로 변환하는 **텍스트 임베딩(Text Embedding)** 기술은 매우 중요합니다. 대중적으로 사용되는 토픽 모델링 기법인 BERTopic에서는 관습적으로 Mean 풀링을 사용하여 문서 임베딩을 생성해 왔으나, 모델의 구조(Encoder vs. Decoder)와 풀링 전략의 조합에 따른 성능 비교 연구는 부족한 실정입니다.

본 연구는 다양한 공개 데이터셋 환경에서 **임베딩 모델군(S-BERT, LLaMA)**과 **단일/듀얼 풀링 기법**의 조합을 교차 평가하여, 최적의 토픽 모델링 성능을 도출할 수 있는 아키텍처 가이드라인을 제시합니다.
